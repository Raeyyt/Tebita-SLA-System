# System Prerequisites

To run the Tebita SLA System locally, you need to install the following two applications on your computer:

## 1. Python (for the Backend)
- **Version**: 3.12 or newer
- **Download**: [python.org/downloads](https://www.python.org/downloads/)
- **Important**: During installation, check the box that says **"Add Python to PATH"**.

## 2. Node.js (for the Frontend)
- **Version**: 18 (LTS) or newer
- **Download**: [nodejs.org](https://nodejs.org/en/download/)
- **Note**: This includes `npm`, which is required to run the frontend.

## Verification
To check if they are installed correctly, open PowerShell and run:
```powershell
python --version
node --version
npm --version
```
